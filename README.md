## ⚡️ Quick start

1. Clone or fork the repository:

- `git clone git@github.com:rubbenpad/podcaster.git`

2. Navigate into your new folder and install dependencies 🔍

- `cd podcaster && npm install`

3. Development ⛏

- `npm run dev`

4. Production 🔥

- `npm run build && npm run start`
